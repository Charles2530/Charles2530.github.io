通用缩写

本页单独撷取了部分极其常用的缩写及相关缩写片段

缩写 全称 备注
pa physical address
va virtual address
tf trap frame
asid address identifier
pgdir page_directory
off offset
ident identifier

vprintfmt variadic print format include/print.h
