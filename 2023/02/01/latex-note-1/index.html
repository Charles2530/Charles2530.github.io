latex_basic

课程代码

课程内容细节：

  1. letter不含有部分。
  2. 对于文章中需要修改的部分，建议使用直接注释掉的方法。
  3. 在latex中可以使用$$包含数学公式进行编写，同时可以使用双​$$进行公式书写并换行。(单表示行间公式)
  4. 如果需要在两段文字间换行，可以在两段文字间加入一行空行，且多个空行在pdf中依然只表示一个换行符。（这行空行不可以写注释，影响换行）
  5. 一个latex有且只有一个document环境。
  6. latex在导言区进行全局设置，在正文区进行文稿编写。

latex中的中文处理方法

课程代码

课程内容细节：

  1. 要想编写中文，首先要保证latex原文件为UTF-8编码。并引入宏包（）
  2. \newcommand用于定义新命令,如示例中度数的定义。
  3. 在写中文时可以在前面定义所写汉字的字体等信息。
  4. 示例中的equation环境用于产生带编号的公式。
  5. 在cmd输入texdoc +宏包名可以查阅相关的宏包说明手册。（lshort-zh是latex的入门说明文件，也可以通过这种方式访问）
  6. 可以使用代替之前的宏包引入。

Latex的字体设置

课程代码

课程内容细节：

  1. 可以用大括号进行分组，以限制字体声明的范围。
  2. \quad为空格，\为换行
  3. 可以在documentclass设置文档参数设置字体大小，正常只有10,11,12pt
  4. 建议使用定义newcommand来设置字体相关参数
